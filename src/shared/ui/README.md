# Shared UI Components

This folder will contain reusable UI components that can be used across the entire application.

## Structure

- `Button/` - Button components with variants
- `Input/` - Input field components
- `Modal/` - Modal and dialog components
- `Form/` - Form-related components
- `Layout/` - Layout components (Header, Footer, Sidebar)
- `Typography/` - Text and typography components
- `Icons/` - Icon components and library

## Usage

Shared UI components should be:

- Framework-agnostic and reusable
- Well-documented with Storybook (future)
- Tested with unit tests
- Following design system guidelines

Import these components when building features and widgets.

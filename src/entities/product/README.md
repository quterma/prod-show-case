# Product Entity

This folder contains the product entity according to Feature-Sliced Design (FSD) architecture.

## Structure

- `model/` - Types, interfaces, schemas, and business logic
- `api/` - API client methods for product operations
- `lib/` - Utility functions and helpers specific to products
- `ui/` - Reusable UI components for product display

## Usage

The product entity provides:

- Product data types and validation schemas
- API methods for CRUD operations
- Shared UI components (ProductCard, ProductInfo, etc.)
- Product-specific utilities and constants

Import from this entity when you need basic product functionality across features and widgets.
